The persistent map picker at the top of the Maps page. Selected pill fills mint; optional per-map win% turns the row into a strength glance. Controlled.

```jsx
<MapPills value={map} onChange={setMap}
  winRates={{ Mirage:61, Inferno:48, Nuke:39, Ancient:55 }} />
```

Persist the selection to localStorage at the call site — the page should remember the last map.
