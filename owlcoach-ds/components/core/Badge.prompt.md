Status / category token in tracked uppercase Chakra Petch. Tone carries meaning — `good` (mint), `bad` (orange), `warn`, `info`, `neutral`.

```jsx
<Badge tone="good" variant="soft">On target</Badge>
<Badge tone="bad" icon={<i data-lucide="trending-down" />}>Regressing</Badge>
```

Variants: `soft` (tint fill, default), `solid` (full accent), `outline`. Keep labels to 1–2 words.
